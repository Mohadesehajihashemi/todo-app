# To-Do List Web App

## توضیح پروژه
این پروژه یک وب‌اپلیکیشن ساده است که به کاربران این امکان را می‌دهد که نظرات خود را وارد کرده، مشاهده کنند و نظرات قدیمی را حذف کنند. نظرات در مرورگر کاربر ذخیره می‌شوند تا پس از بارگذاری مجدد صفحه همچنان باقی بمانند. این پروژه با استفاده از **HTML**، **CSS** و **JavaScript** ساخته شده است.

## ویژگی‌ها
- **اضافه کردن نظر**: کاربران می‌توانند نظرات خود را وارد کنند.
- **ذخیره‌سازی نظرات**: نظرات به صورت محلی در **localStorage** مرورگر ذخیره می‌شوند.
- **حذف نظرات**: کاربران می‌توانند نظرات خود را حذف کنند.
- **حفظ داده‌ها پس از بارگذاری صفحه**: نظرات بعد از بارگذاری مجدد صفحه همچنان باقی می‌مانند.

## نحوه استفاده
1. پروژه را از گیت‌هاب دانلود یا کلون کنید.
2. فایل `index.html` را در مرورگر خود باز کنید.
3. نظرات خود را وارد کنید و پس از ارسال، آن‌ها را مشاهده خواهید کرد.
4. برای حذف یک نظر، روی دکمه "حذف" کنار آن کلیک کنید.
